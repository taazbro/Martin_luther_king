# Community Packs

This folder is the open-source starter layer for EduClawn.

It contains:

- `sample_projects/`: ready-made starter manifests
- `school_packs/`: installable starter packs for districts, nonprofits, and teachers
- `plugins/`: local extension examples
- `template_starters/`: starter manifests for template authors

## Install A Pack And Start

1. Pick a pack from the Marketplace page or call the school-pack install endpoint.
2. Open the generated sample under `community/sample_projects/`.
3. Copy the starter into a classroom or project workflow.
4. Add local documents, run compile, and export.
